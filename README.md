# my-toodledo

user-script for toodledo

Feature:

* add Gmail like shortcut
    * Move: j/k
    * Navigate: m/s/c
    * Action:
        * **t**ask
        * **f**older
        * **c**ontext
        * pr*i*ority
        * t**i**mer
        * **l**ength
        * enter = timer start/pause
        * x = done/undo
