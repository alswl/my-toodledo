# my-toodledo

user-script for toodledo

Feature:

add Gmail like shortcut:

*   Move: j/k
*   Navigate: m/s/c
*   Action:
    *   t = **t**ask
    *   f = **f**older
    *   c = **c**ontext
    *   i = pr**i**ority
    *   i = t**i**mer
    *   l = **l**ength
    *   enter = timer start/pause
    *   x = done/undo
    *   D = **d**uplicate
    *   X = delete
    *   S = add **s**ub task


Works well for Chrome and Safari.
